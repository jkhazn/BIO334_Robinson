# BIO334 Practical Bioinformatics

University of Zurich — Spring 2026

Analysis of complex bioinformatic data using Python and R.

**Note:** This repository is named `BIO334_Robinson` after the final professor block but covers all four blocks of the course.

## Course Structure

| Block | Instructor | Dates | Topics |
|-------|-----------|-------|--------|
| 1 | von Mering | May 7, 8, 12 | Python basics, multiple sequence alignment, NJ trees |
| 2 | Wagner | May 13, 15, 19 | Fitness landscapes, DHFR data, igraph |
| 3 | Shimizu | May 20, 21, 22 | Nucleotide diversity, GFF annotation, Tajima's D |
| 4 | Robinson | May 22 onward | R, RNA-seq, data analysis automation |
